.env Configuration

Copy the variables below into your .env file at the project root, then restart the server.

# TheHive SOAR
THEHIVE_URL=http://localhost:9000
THEHIVE_API_KEY=your-thehive-api-key
THEHIVE_ENABLED=true

# Cortex
CORTEX_URL=http://localhost:9001
CORTEX_API_KEY=your-cortex-api-key
CORTEX_ENABLED=true

# Wazuh SIEM
WAZUH_URL=https://localhost:55000
WAZUH_USER=wazuh
WAZUH_PASSWORD=wazuh
WAZUH_ENABLED=true

# Threat Intel
VIRUSTOTAL_API_KEY=your-vt-key
VIRUSTOTAL_ENABLED=true
# AI Engine — Claude (recommended)
ANTHROPIC_API_KEY=your-anthropic-key
ANTHROPIC_MODEL=claude-sonnet-4-6
# OR OpenAI fallback
# OPENAI_API_KEY=your-openai-key
# OPENAI_MODEL=gpt-4o-mini
# AI_ENABLED=true
